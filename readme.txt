This plugin will email comment reply from Blog/Site admin as an email.

== Description ==

* Blog user will be notified by admin reply via email
* Email sent address will be admin's email address

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

1.How I can add use any other email address rather than using admin email?

Answer:Right now this feature is not available, we will implement this feature in next version.

2.Does this plugin email only subscribed users? 
Answer: No,Its email all users who got reply in their comment.
